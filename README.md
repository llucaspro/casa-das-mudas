# 🌿 Casa das Mudas

  > Mudas cultivadas com cuidado em Tupã, SP

  Site multi-página para a Casa das Mudas, desenvolvido em HTML, CSS e JavaScript puro.

  ## Páginas

  | Página | Descrição |
  |--------|-----------|
  | [index.html](index.html) | Página inicial — hero, produtos em destaque, testemunhos |
  | [mudas.html](mudas.html) | Catálogo completo com 12 variedades e filtros por categoria |
  | [sobre.html](sobre.html) | Nossa história, linha do tempo, valores e galeria |
  | [contato.html](contato.html) | Formulário de contato, informações e FAQ |

  ## Funcionalidades

  - Animações de scroll (reveal-on-scroll com IntersectionObserver)
  - Hero com zoom cinematográfico e partículas flutuantes
  - Spine decorativa que acompanha o scroll
  - Carrinho persistente com localStorage
  - Filtros de catálogo por categoria
  - Contadores animados (count-up)
  - Menu mobile responsivo
  - Toast de notificação
  - Formulário de contato com validação
  - FAQ com acordeão nativo (`<details>`)
  - Design responsivo para todos os dispositivos

  ## Stack

  - HTML5 semântico
  - CSS3 (variáveis, grid, animações, transitions)
  - JavaScript vanilla (sem dependências)
  - Google Fonts: Bricolage Grotesque, Inter, IBM Plex Mono
  - Imagens: Unsplash

  ## Configurações

  Para usar o site, edite:
  - Número de WhatsApp em `contato.html`
  - E-mail em `contato.html` e no footer de todas as páginas
  - Preços e variedades em `mudas.html`

Conectar ao GitHub

Para salvar produtos e fotos, o painel precisa de um Token de Acesso do GitHub. Você só precisa configurar isso uma vez — fica guardado neste navegador.

🔑 Como gerar seu token: 1. Acesse github.com/settings/tokens
2. Marque a permissão repo
3. Clique em "Generate token" e cole abaixo
Token inválido ou sem acesso ao repositório.

O token fica salvo apenas neste navegador. Se trocar de dispositivo, basta repetir este passo.
